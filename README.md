# Sachin Eldho — Portfolio

A cyberpunk, terminal-inspired portfolio for **Sachin Eldho**, AI & full-stack developer.
Features interactive CLI-style navigation, a draggable in-browser terminal, CRT/scanline
effects, and liquid-glass project pages.

Built with React 19, TypeScript, React Router, Vite, and Tailwind CSS.

## Run Locally

**Prerequisites:** Node.js 18+

1. Install dependencies:
   `npm install`
2. Run the app:
   `npm run dev`
3. Build for production:
   `npm run build`
